# AI Cricket Squad Selector (Frontend)

A React + Vite + Tailwind app for generating cricket squads and viewing player details.

## Prerequisites
- Node.js v18+ and npm

## Getting Started
```bash
npm install
npm run dev
# open http://localhost:8080/
```

## Build & Preview
```bash
npm run build
npm run preview
```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Configuration
- Dev server port can be updated in `vite.config.ts` (default `8080`).
- Aliases configured at `@` → `src`.

## Folder Structure
- `src/pages` — screens and routes
- `src/components` — UI components
- `src/lib` — helpers and mock data
- `public` — static assets
